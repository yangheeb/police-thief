# 🚨 Police & Thief Simulation

멀티스레드 기반 경찰-도둑-금고 시뮬레이션 프로그램입니다.  
여러 스레드가 동시에 동작하며 공유 자원(금고)에 접근할 때 발생하는  
동시성 문제를 제어하고 로직을 구현하는 것을 목표로 합니다.

---

## 프로젝트 개요

- 도둑, 경찰, 금고, 리포터가 각각 스레드로 동작
- 금고는 공유 자원으로 동시 접근 제어 필요
- 일정 시간 동안 시뮬레이션 진행 후 결과 출력

---

## 구성 요소

### 💰 금고
- 위치: (10, 10) 고정
- 초기 잔액: 100,000원
- 역할:
  - 도둑이 훔친만큼 총액에서 감소
  - 도둑이 체포되면 금액 복구
- 동시성 제어:
  - synchronized 적용

---

### 🥷 도둑
- 여러 명 존재
- 일정 주기로 랜덤 이동
- 금고 도착 시 랜덤 금액 훔침
- 체포 시 활동 중지

---

### 👮🏻‍♂️ 경찰
- 여러 명 존재
- 가장 가까운 도둑 추적
- 체포 시 도둑이 훔친 금액 회수
- 체포 횟수 집계

---

### 💸 리포터
- 일정 주기마다 상태 출력
- 출력 정보:
  - 금고 잔액
  - 총 도난액
  - 총 회수액
  - 체포 횟수

---

## ⚙ 기술 스택

- Language: Java
- Build Tool: Maven
- Thread: Java Thread API
